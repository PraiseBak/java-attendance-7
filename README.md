# 출석 시스템
- 출석 관련 날짜 관리 기능
  - 시스템에 저장된 시간을 기준으로함
  - 월 13 ~ 18
  - 나머지 10 ~ 18
  - 5분초과 지각, 30분 초과 결석, 출석기록 없는 날 결석
- 누적 지각 처리
  - 결석횟수 5회 초과시 제적
  - 지각 3회는 결석 1회로 간주한다.
    경고 대상자: 결석 2회 이상
    면담 대상자: 결석 3회 이상
    제적 대상자: 결석 5회 초과
    캠퍼스 운영 시간은 매일 08:00~23:00이다.
    주말 및 공휴일에는 출석을 받지 않는다.

- 출석 확인
  - 닉네임, 등교 시간 입력
  - 날짜 출력해줌
- 출석 수정
  - 닉네임,수정날짜,변경전,후 출력 가능

- 출석 기록 확인 가능 - 닉네임으로 입력

- 제적 위험자
  - 출석 기록으로 제적 위험자 판단 하여 출력(출석 기록 기반 정렬, 닉네임 오름차순, 정렬)
